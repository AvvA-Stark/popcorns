# 🍿 Popcorns

## Vision
Tinder for movies meets Trustpilot with streaming availability.
Solve the "what should I watch tonight?" problem.

## Reference
JustWatch.com — but with swipe discovery + real user reviews

## Tech Stack
- **Frontend:** React Native + Expo (iOS primary, web secondary)
- **Backend:** Supabase (PostgreSQL + Auth + Realtime)
- **Movie Data:** TMDB API
- **Streaming Info:** TMDB Watch Providers
- **Hosting:** VPS (API) + App Store (iOS)

## MVP Features
1. 🎬 Swipe Discovery — Tinder-style card stack
2. ⭐ Real Reviews — User-written reviews + ratings
3. 📺 Where to Watch — Streaming availability by region
4. 🎯 Smart Recommendations — Learn from swipes
5. 👤 User Profiles — Watchlist, reviewed movies
6. 🔔 Push Notifications — New matches on your platforms
